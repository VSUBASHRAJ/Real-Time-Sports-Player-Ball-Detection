# Real-Time Sports Player & Ball Detection
 
**Technologies:** Python, PyTorch, YOLOv8, OpenCV  

---

## Project Overview

This project implements a **Real-Time Sports Player & Ball Detection System** using **PyTorch** and **YOLOv8**. It detects players and balls in videos or live streams and can also display other objects present in the scene. The project is designed for sports analytics, match analysis, and computer vision enthusiasts.

Key Features:

- Real-time detection of players and sports balls in video streams.  
- Highlight other objects in the scene (optional).  
- Bounding box visualization with object labels.  
- Fully implemented in Python using PyTorch and YOLOv8.  

---

## Dataset

This project uses pre-trained YOLOv8 models. No custom training is required.  

- **Sample Video:** Users can upload their own sports video (e.g., soccer, basketball).  
- **Objects:** Person (player), sports ball, and other objects detected using YOLOv8 pre-trained COCO classes.
